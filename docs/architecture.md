# CryptoScanPS — Architecture

## Overview
CryptoScanPS — минималистичный сканер крипторынка на PowerShell.

Основные элементы:
- `src/CryptoScan.ps1` — точка входа
- `src/modules/*.psm1` — модульная логика (HTTP, парсер, утилиты)
- `src/config.json` — конфигурация (биржа, символы, фильтры)
- `samples/` — примеры вывода и логов

## Поток данных
1. Скрипт читает `config.json`
2. Для каждого символа вызывается модуль `Get-DemoTicker` (в будущем — реальный API)
3. Данные приводятся к единому формату через `Parse-DemoTicker`
4. Результаты фильтруются по объёму
5. Итог выводится в JSON и логируется в текстовый файл
