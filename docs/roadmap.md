# CryptoScanPS — Roadmap

## v1.0
- Базовый сканер на демо-данных
- Конфиг через `config.json`
- Логирование в файл
- Примеры вывода (`samples/`)

## v1.1 (планы)
- Реальные HTTP-запросы к API биржи
- Обработка ошибок и ретраи
- Дополнительные фильтры (изменение цены, волатильность)
- Telegram-нотификации (через бота)

## v1.2+ (идеи)
- Поддержка нескольких бирж
- Интеграция с "PulseAgent"
- CLI-параметры для быстрого запуска пресетов
